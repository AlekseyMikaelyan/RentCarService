Project web app for renting cars.
------------------------------------------------------------------------------------------------
This application is developed using MAVEN, HIBERNATE, SPRING BOOT, POSTGRESQL, H2, APACHE TOMCAT
------------------------------------------------------------------------------------------------
The application consists of 6 modules:
1 entity;
2 dto;
3 repository;
4 service;
5 controller;
6 exceptions.
------------------------------------------------------------------------------------------------
The application uses the main postgresql database. The ability to perform basic CRUD operations
on entities from database.
We can interact with the application through POST, PUT, GET, DELETE requests.
------------------------------------------------------------------------------------------------
In this project the methods of the service and controller layers were tested using libraries
Mockito and JUnit.
------------------------------------------------------------------------------------------------
Integration testing was also performed in this project. H2 database uses for tests.
------------------------------------------------------------------------------------------------
All dependencies spelled out in the main pom.xml file in the root folder.
